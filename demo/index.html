Welcome to the simpler-sidebar development playground.

Press the menu icon at the right corner of the page to navigate through this playground.

This section of the simpler-sidebar website is used to test hotfix and pre-releases, so if you encounter any problems, go back to the main website.

This playground is delivered with the simpler-sidebar plugin, you can find it in the demo directory.

If you need a guide to install the plugin, you are in the wrong place. Check the Getting Started guide in the main website or directly on the simpler-sidebar github page at this link.
